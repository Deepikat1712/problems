# Day 04 - LeetCode Practice

## Problem 1

### Convert the Temperature

- **Problem Number:** 2469
- **Platform:** LeetCode
- **Difficulty:** Easy
- **Language:** Java

### Concepts Learned

- Variables
- Data Types (`double`)
- Arithmetic Operators
- Arrays
- Return Statement

### Problem Summary

Given a temperature in Celsius, convert it to Kelvin and Fahrenheit, then return both values as an array.

### Formulas Used

- Kelvin = Celsius + 273.15
- Fahrenheit = Celsius × 1.80 + 32.00

### Key Learning

- Working with `double` data type
- Creating and returning an array
- Using arithmetic expressions
- Understanding Java's case-sensitive variable names
